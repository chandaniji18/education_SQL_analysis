# 📊 Education SQL Analysis Project

## 📌 Project Overview
This project analyzes district-level student enrollment data using SQL.  
The objective is to extract meaningful insights from raw education data using queries and aggregations.

---

## 🛠 Tools & Technologies Used
- MySQL
- SQL (Joins, Aggregations, Group By, Order By)
- GitHub

---

## 📂 Dataset Information
The dataset contains district-level data including:
- Total students
- Boys count
- Girls count
- Enrollment percentages
- School categories (Primary, Middle, Secondary)

---

## 🔎 Key Analysis Performed
- Total number of students calculation
- District-wise comparison
- Boys vs Girls distribution
- Average enrollment percentage
- Ranking districts based on performance
- Category-wise (Primary/Middle/Secondary) insights

---

## 📈 Key Insights
- Student distribution varies significantly across districts.
- Some districts show higher female enrollment percentages.
- Secondary level enrollment is lower compared to Primary level in several districts.
- Data highlights areas that may require educational improvement focus.

---

## 📁 Files Included
- `education_project.sql` → All SQL queries with answers and observations.

---

## 🚀 How to Run
1. Import the dataset into MySQL.
2. Open the `education_project.sql` file.
3. Execute queries to view results and insights.
